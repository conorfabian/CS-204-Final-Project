EXPERIMENT CHECKLIST

Before:
- Close unnecessary apps
- Clear Chrome cache
- Create data/raw/trial_XXX/ directory
- Have stopwatch ready

Setup:
- Run ./scripts/collection/launch_chrome.sh
- Load YouTube video (paused)
- Open chrome://media-internals
- Run ./scripts/network/manual_control.sh, select HIGH

During (135s):
- T=0: Start video
- T=45: Switch to LOW
- T=90: Switch to HIGH
- T=135: Stop video

After:
- Export media-internals data
- Run ./scripts/network/cleanup.sh
- Record observations in trial folder
